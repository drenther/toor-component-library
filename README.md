# Hosted @ - https://toor-design.surge.sh

Components for Toor

## USAGE

_Start by cloning the repo_

```
git clone https://github.com/drenther/toor-component-library.git
```

_Then Install the npm dependencies_

```
npm install
```

_To start dev server_

```
npm start
```

_To build_

```
npm run build
```
